# Engineering Capability Pack — Phase 4.1

This pack turns the `engineering-core` knowledge layer into small governed workflows:

- `SteelWeightWorkflow` — calculates section/member weight.
- `MaterialTakeoffWorkflow` — aggregates member and plate quantities.
- `PlateOptimizationWorkflow` — estimates stock plate utilisation and waste.
- `SectionSelectionWorkflow` — selects starter catalog sections by family/area/weight.
- `ConnectionSizingWorkflow` — sizes a simple bolt/anchor connection block.
- `CodeValidationWorkflow` — validates selected code and demand/capacity checks.

The pack is intentionally lightweight and does not perform full structural design yet.
