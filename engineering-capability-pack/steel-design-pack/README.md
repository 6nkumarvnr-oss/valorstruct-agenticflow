# Steel Design Pack MVP — Phase 4.3

This is a preliminary engineering-check capability pack for early AgenticFlow demos.

It uses simple educational demand/capacity ratios for:

- axial tension
- axial compression
- bending
- shear
- deflection

The workflow uses `engineering-core` for section/material lookup and `engineering-science-core` mechanics formulas where possible. It is **not** a final code-compliant design engine.
