# Valor Quotation Capability Pack MVP

This first commercial capability pack proves a practical engineering-business workflow:

```text
Project Input → BOQ Items → Rate Calculation → Quotation Summary → Simple Structural Calculation → Report Output → Audit Log → P-Agent Memory Entry
```

It intentionally does not perform full structural design or PDF generation yet. The pack provides reusable blocks for future no-code AgenticFlow workflows:

- Input Block
- BOQ Block
- Rate Calculation Block
- Simple Calculation Block
- Report Block
- Memory Block
