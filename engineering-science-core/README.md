# Engineering Science Core

The Engineering Science Core is the scientific foundation below `engineering-core` and `engineering-capability-pack`.

```text
engineering-science-core/
├── mathematics/          algebra, calculus, linear algebra, numerical methods, statistics, optimization
├── physics/              kinematics, dynamics, statics, thermodynamics, fluid mechanics, electromagnetics
├── mechanics/            stress, strain, beam theory, column theory, buckling, fatigue, fracture, vibration
├── structural-analysis/  truss, frame, space frame, plate, shell, connection
├── optimization/         weight, cost, member selection, layout optimization
├── simulation/           load, traffic, fabrication, construction simulation
└── ai-reasoning/         engineering knowledge graph primitives
```

This is not a replacement for STAAD, SAP2000, or ETABS. It is the first typed science layer for future governed engineering agents.

## Phase 4.2 Focus

Phase 4.2 strengthens the four science modules that future engineering agents will use most:

1. `mathematics/` for algebra, calculus, linear algebra, numerical methods, statistics, and optimization.
2. `physics/` for kinematics, dynamics, and statics foundations.
3. `mechanics/` for stress, strain, beam theory, column theory, buckling, and vibration primitives.
4. `structural-analysis/` for reaction solving, frame stiffness, and connection load-path reasoning.
