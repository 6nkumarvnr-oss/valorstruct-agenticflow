# AgenticFlow / ValorStruct Agentic Architecture

AgenticFlow is a governance-driven agentic operating system for technical business workflows.

## Core stack

```text
PatchD Governance Organ
P-Agent Execution Runtime
Workflow Orchestration Core
AI Gateway + Model Role Routing
Governed Capability Specialists
Governed Swarm Reasoning Protocol
```

## Product objective

Convert engineering, manufacturing, drawing, quotation, business, and software requests into governed, auditable, human-approved packages.

## MVP user types

```text
Admin / owner
Senior engineer approver
Commercial approver
Operator / estimator
Client reviewer
```

## MVP modules

```text
Agent registry
Capability registry
Capability contracts
Workflow orchestration
Governed swarm run
Audit trail
Human approval packet
Frontend swarm console
Backend swarm endpoint
```

## Human approval points

Approval is required for:

```text
engineering design release
quotation release
manufacturing package release
code deployment
IAM/security changes
legal/compliance-sensitive output
public release
paid provider retry at high cost
```

## AI system rule

AI may draft, classify, critique, synthesize, explain, and route.

AI must not independently approve engineering release, quotation issue, legal advice, deployment, or public release.

Software architecture draft prepared for human review and approval.
